## About The Project
Simple boid simulator with customizable parameters such as:  
* Amount/Size.
* Factors affecting how extreme their motion is.
* Behaviour
  
If you dont know what boids are: [Wikipedia](https://en.wikipedia.org/wiki/Boids).
